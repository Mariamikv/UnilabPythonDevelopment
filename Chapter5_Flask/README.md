рЃўрЃюрЃбрЃћрЃарЃюрЃћрЃб рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃўрЃЌ рЃџрЃЮрЃЎрЃљрЃџрЃБрЃа рЃАрЃћрЃарЃЋрЃћрЃарЃЌрЃљрЃю рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃўрЃАрЃљрЃА (рЃЕрЃћрЃЏрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃќрЃћ: http://127.0.0.1:5000/), рЃАрЃљрЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃЮ рЃЋрЃћрЃџрЃерЃў рЃњрЃљрЃЏрЃЮрЃЋрЃљ рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃў "рЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ"

### REST-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў
#### HTTP Verbs - рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃА рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў

рЃАрЃћрЃарЃЋрЃћрЃарЃБрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃцрЃБрЃюрЃфрЃЦрЃўрЃўрЃљ рЃЏрЃўрЃўрЃдрЃЮрЃА рЃЋрЃћрЃЉ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃў.

рЃЋрЃћрЃЉ рЃАрЃљрЃўрЃбрЃќрЃћ рЃерЃћрЃАрЃЋрЃџрЃўрЃАрЃљрЃА, рЃАрЃћрЃарЃЋрЃћрЃарЃќрЃћ рЃњрЃљрЃерЃЋрЃћрЃЉрЃБрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃўрЃА рЃцрЃљрЃюрЃ»рЃљрЃарЃљрЃерЃў рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ (рЃџрЃЮрЃњрЃў) рЃерЃћрЃЏрЃЮрЃАрЃБрЃџрЃў рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ, рЃЏрЃљрЃњ.:

127.0.0.1 - - [30/Mar/2020 01:31:33] "GET / HTTP/1.1" 200 -

рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџрЃў рЃџрЃЮрЃњрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ рЃерЃћрЃЏрЃЊрЃћрЃњрЃюрЃљрЃўрЃарЃўрЃљ:

рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў - - [рЃЌрЃљрЃарЃўрЃдрЃў(рЃЊрЃдрЃћ/рЃЌрЃЋрЃћ/рЃгрЃћрЃџрЃў рЃАрЃљрЃљрЃЌрЃў:рЃгрЃБрЃЌрЃў:рЃгрЃљрЃЏрЃў)] "рЃбрЃўрЃърЃў  рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў  рЃърЃарЃЮрЃбрЃЮрЃЎрЃЮрЃџрЃў" рЃАрЃбрЃљрЃбрЃБрЃАрЃў -

**рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃА рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃўрЃА рЃбрЃўрЃърЃћрЃЉрЃў:**

**GET** - рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃўрЃА рЃљрЃЏрЃЮрЃдрЃћрЃЉрЃљ
**POST** - рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ
**PUT** - рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃљрЃю рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ
**DELETE** - рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃўрЃА рЃљрЃЏрЃЮрЃерЃџрЃљ

рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃЏрЃ«рЃљрЃарЃћрЃА рЃДрЃЋрЃћрЃџрЃљ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃА рЃбрЃўрЃърЃў рЃћрЃарЃЌрЃюрЃљрЃўрЃарЃљрЃЊ рЃЏрЃБрЃерЃљрЃЋрЃЊрЃћрЃЉрЃљ. рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃБрЃџрЃў рЃбрЃўрЃърЃћрЃЉрЃў рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃўрЃА рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљрЃАрЃљ рЃЊрЃљ рЃњрЃљрЃарЃЕрЃћрЃЋрЃљрЃерЃў рЃњрЃЋрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ. рЃћрЃарЃЌ рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃќрЃћ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃарЃљрЃЏрЃЮрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃбрЃўрЃърЃўрЃА рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ, рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:

**"GET /item/student"** - рЃарЃћрЃАрЃБрЃарЃАрЃўрЃА рЃљрЃЏрЃЮрЃдрЃћрЃЉрЃљ

**"POST /item/student + рЃърЃљрЃбрЃљрЃарЃЏрЃћрЃбрЃарЃў"** - рЃарЃћрЃАрЃБрЃарЃАрЃўрЃАрЃЌрЃЋрЃўрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃА рЃЏрЃўрЃгрЃЮрЃЊрЃћрЃЉрЃљ

**"PUT /item/student + рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў"** - рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ рЃарЃћрЃАрЃБрЃарЃАрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃљрЃю рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ

**"DELETE /item/student"** - рЃарЃћрЃАрЃБрЃарЃАрЃўрЃА рЃљрЃЏрЃЮрЃерЃџрЃљ

рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃерЃў, /course/student рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃќрЃћ рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃЏрЃљрЃюрЃўрЃърЃБрЃџрЃўрЃарЃћрЃЉрЃљ рЃўрЃюрЃЊрЃўрЃЋрЃўрЃЊрЃБрЃљрЃџрЃБрЃа рЃарЃћрЃАрЃБрЃарЃќрЃћ(item resource). рЃЌрЃБрЃЏрЃфрЃљ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃА рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃљ рЃљрЃюрЃљрЃџрЃЮрЃњрЃБрЃарЃљрЃЊ рЃљрЃарЃўрЃА рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃАрЃўрЃљрЃќрЃћрЃф (itemList resource) рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃќрЃћ /item.

REST рЃбрЃўрЃърЃўрЃА рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ рЃљрЃарЃўрЃА **Stateless** рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ, рЃарЃљрЃф рЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃЮрЃЉрЃА рЃарЃЮрЃЏ рЃћрЃарЃЌрЃў рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ рЃЋрЃћрЃа рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃў рЃЏрЃћрЃЮрЃарЃћ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљрЃќрЃћ, рЃарЃљрЃЊрЃњрЃљрЃю рЃЏрЃљрЃЌ рЃерЃЮрЃарЃўрЃА рЃљрЃа рЃљрЃарЃўрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃарЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃў рЃЎрЃљрЃЋрЃерЃўрЃарЃў. рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃљрЃЊ, рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃўрЃАрЃљрЃА, рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃўрЃарЃћрЃЉрЃА рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ. 


### App Endpoints - рЃарЃћрЃАрЃБрЃарЃАрЃўрЃА рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃћрЃЉрЃў

рЃарЃћрЃАрЃБрЃарЃАрЃўрЃА рЃбрЃўрЃърЃћрЃЉрЃў рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃАрЃљ рЃЊрЃљ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃўрЃАрЃЌрЃЋрЃўрЃА рЃБрЃарЃЌрЃўрЃћрЃарЃЌрЃерЃћрЃЉрЃарЃБрЃюрЃћрЃЉрЃБрЃџ рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃљрЃА рЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃА:

**рЃбрЃўрЃърЃў - рЃцрЃБрЃюрЃфрЃЦрЃўрЃЮрЃюрЃљрЃџрЃў рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃАрЃЌрЃЋрЃўрЃА - рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃўрЃАрЃЌрЃЋрЃўрЃА**

POST - рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ - рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃўрЃА рЃЏрЃўрЃгрЃЮрЃЊрЃћрЃЉрЃљ
GET - рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃўрЃА рЃЊрЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃљ - рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ

Flask-рЃўрЃЌ рЃарЃћрЃАрЃБрЃарЃАрЃўрЃА рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃА рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ рЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ рЃарЃЮрЃБрЃЌрЃћрЃЉрЃА. рЃарЃЮрЃБрЃЌрЃўрЃА рЃњрЃљрЃАрЃљрЃЏрЃљрЃарЃЌрЃљрЃЊ рЃњрЃЋрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ рЃЏрЃўрЃЋрЃБрЃЌрЃўрЃЌрЃЮрЃЌ рЃарЃћрЃАрЃБрЃарЃАрЃўрЃА рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў,  рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ рЃАрЃљрЃЎрЃџрЃљрЃАрЃЮ рЃЮрЃЌрЃљрЃ«рЃќрЃћ рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃА рЃњрЃљрЃАрЃљрЃЏрЃљрЃарЃЌрЃљрЃЊ рЃЋрЃгрЃћрЃарЃЌ:


```python
@app.route('/classroom', methods = ['POST'])
```

рЃћрЃарЃЌрЃў рЃарЃЮрЃБрЃЌрЃќрЃћ рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃА рЃЊрЃљрЃерЃЋрЃћрЃЉрЃљ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃбрЃўрЃърЃўрЃА рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃўрЃЌ. рЃљрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА methods-рЃА рЃБрЃюрЃЊрЃљ рЃЏрЃўрЃЋрЃљрЃюрЃўрЃГрЃЮрЃЌ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў, рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ: methods = ['POST', 'GET']

рЃарЃЮрЃБрЃЌрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃЏрЃћрЃЌрЃЮрЃЊрЃўрЃА рЃњрЃљрЃгрЃћрЃарЃљ. рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:


```python
@app.route('/classroom', methods = ['POST'])

    # рЃЎрЃЮрЃЊрЃў рЃЉрЃљрЃќрЃљрЃерЃў рЃАрЃљрЃЎрЃџрЃљрЃАрЃЮ рЃЮрЃЌрЃљрЃ«рЃўрЃА рЃЊрЃљрЃАрЃљрЃЏрЃљрЃбрЃћрЃЉрЃџрЃљрЃЊ
    
    return 200, 'ok'
```

200 ok рЃљрЃарЃўрЃА рЃАрЃбрЃљрЃбрЃБрЃА рЃЎрЃЮрЃЊрЃў рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃўрЃА рЃЏрЃ«рЃљрЃарЃћрЃА рЃљрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃА рЃарЃЮрЃЏ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃљ. (рЃцрЃџрЃљрЃАрЃЎрЃерЃў рЃўрЃюрЃбрЃћрЃњрЃарЃўрЃарЃћрЃЉрЃБрЃџ рЃАрЃбрЃљрЃбрЃБрЃА рЃЎрЃЮрЃЊрЃћрЃЉрЃќрЃћ рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ: https://www.flaskapi.org/api-guide/status-codes/)

flask-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ, рЃгрЃўрЃюрЃљрЃАрЃгрЃљрЃа рЃњрЃљрЃгрЃћрЃарЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ, рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЏрЃўрЃЋрЃгрЃЋрЃЊрЃћрЃЌ рЃарЃћрЃАрЃБрЃарЃАрЃўрЃА рЃАрЃўрЃљрЃерЃў рЃљрЃарЃАрЃћрЃЉрЃБрЃџ рЃљрЃю рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃљрЃЊ рЃЕрЃљрЃЏрЃљрЃбрЃћрЃЉрЃБрЃџ рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃа рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃА <string:name> рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃўрЃЌ, рЃарЃЮрЃЏрЃџрЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљрЃф рЃЏрЃћрЃЌрЃЮрЃЊрЃА рЃњрЃљрЃЊрЃљрЃћрЃфрЃћрЃЏрЃљ рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃљрЃЊ. рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:

рЃљрЃЏ рЃарЃЮрЃБрЃЌрЃА рЃерЃћрЃЋрЃФрЃџрЃћрЃЉрЃЌ рЃЏрЃўрЃЋрЃљрЃЎрЃўрЃЌрЃ«рЃЮрЃЋрЃЌ рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃќрЃћ: <localhost:5000/classroom/pythonclass>

рЃЏрЃћрЃЌрЃЮрЃЊрЃА рЃЎрЃў рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃљрЃЊ рЃњрЃљрЃЊрЃљрЃћрЃфрЃћрЃЏрЃљ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ "pythonclass", рЃарЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљрЃАрЃљрЃф рЃерЃћрЃЋрЃФрЃџрЃћрЃЉрЃЌ рЃЎрЃЮрЃЊрЃўрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃБрЃа рЃюрЃљрЃгрЃўрЃџрЃерЃў. рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃарЃЮрЃЏ рЃарЃЮрЃБрЃЌрЃерЃў рЃњрЃљрЃгрЃћрЃарЃўрЃџ рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃерЃў рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃЊрЃљ рЃЏрЃћрЃЌрЃЮрЃЊрЃерЃў рЃњрЃљрЃгрЃћрЃарЃўрЃџрЃў рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў рЃўрЃДрЃЮрЃА рЃўрЃЊрЃћрЃюрЃбрЃБрЃарЃў.

рЃЏрЃўрЃБрЃ«рЃћрЃЊрЃљрЃЋрЃљрЃЊ рЃўрЃЏрЃўрЃАрЃљ рЃарЃЮрЃЏ рЃЏрЃћрЃЌрЃЮрЃЊрЃўрЃА рЃЊрЃљрЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃљрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃљрЃа рЃљрЃЦрЃЋрЃА, рЃЏрЃљрЃА рЃБрЃюрЃЊрЃљ рЃћрЃарЃЦрЃЋрЃљрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃАрЃљрЃ«рЃћрЃџрЃў.

            return {"address": cinema['address']}

        "name": name,
        "address": request_data['Address']
    }
