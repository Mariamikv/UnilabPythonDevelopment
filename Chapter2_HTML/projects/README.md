## ინსტრუქცია:

### პირველი პროექტი
მოამზადეთ ნებისმიერ html გვერდი შეხვედრაზე განვლილი ელემენტების გამოყენებით. 

აუცილებელია გამოიყენოთ 5 სხვადასხვა ტიპის HTML ელემენტი.
მაგალითად: ```header, paragraph, unordered list, table, image```
